# 🏥 📅 90-DAY NORCET STUDY PLAN (SMART + BALANCED)

## 🔁 Structure

* **Days 1–60 → First reading (slow + strong)**
* **Days 61–75 → 2nd revision (fast + MCQ heavy)**
* **Days 76–90 → Mock tests + final polishing**
