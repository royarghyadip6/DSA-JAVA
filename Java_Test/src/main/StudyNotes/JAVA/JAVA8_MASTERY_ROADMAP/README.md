# Java 8 Mastery Roadmap — Study Guide Index

> **For:** Developers who know Java 8 basics and want **mastery + interview readiness (5–8 years level)**.  
> **Each phase file contains:** Easy theory (bullet points) → Examples → Practice exercises → Senior interview Q&A.

---

## How to Study

| Step | What to Do |
|------|------------|
| 1 | Start with [Phase 0](00-prerequisites.md) if you have gaps in OOP, collections, or threads |
| 2 | Follow phases **1 → 10** in order |
| 3 | After reading theory, **type every example** yourself — don't just read |
| 4 | Solve **all practice exercises** before moving to the next phase |
| 5 | Review **Interview Q&A** at the end of each phase |
| 6 | Keep [API Reference](API-REFERENCE.md) open while coding |

**Daily rule:** Read → Code → Break → Fix → Refactor.
